◆ setFramerate()

void game::Game::setFramerate ( const game::uint8 &  framerate)

Limit the framerate to a maximum fixed frequency. If a limit is set, the window will use a small delay after each call to display() to ensure that the current frame lasted long enough to match the framerate limit. SFML will try to match the given limit as much as it can, but since it internally uses sf::sleep, whose precision depends on the underlying OS, the results may be a little unprecise as well (for example, you can get 65 FPS when requesting 60).

Parameters
volumeNumber x in frames per seconds. (e.g.: 30)
