14 /*
15  Communication between server and clients
16  ----------------------------------------
17 
18  CONNECTION client -> server
19  observe:
20  start a connection, the person will receive all infos of the game
21  disobserve:
22  stop sending game info
23  connect [pseudo]:
24  start a connection, for players
25  pseudo: pseudo that the player wants to be identified with
26  disconn [id]:
27  end a connection
28  id: id of the player in the game
29 
30  END client -> server
31  killServer:
32  kills the server no matter what is the context
33 
34  INFO server -> client
35  infoLobby [time] [connected] [max] [difficulty]
36  information on the current lobby
37  time: time remaining before the start of the game
38  connected: number of players in queue
39  max: max number of players in the game
40  difficulty: difficulty of the game
41  startingGame [connected] [difficulty]
42  connected: number of players in queue
43  difficulty: difficulty of the game
44 
45  MOVE client -> server
46  move id dir
47  move player with id [id] to a direction
48  id: id of the player requesting a move
49  dir: NN, SS, WW EE, NE, NW, SW, or SE (North South West East,
50  North-East, North-West, South-West, South-East)
51 
52  stop id
53  stop player with id [id]
54  id: id of the player requesting a stop
55 
56  Bomb client -> server
57  bomb id
58  Drop a bomb at player with id [id] position
59  id: id of the player requesting to drop a bomb
60 
61  Victory server -> client
62  win
63  Send a message to the winner
64 
65  End server -> client
66  end
67  Send an end message to the client
68 
69  SUCCESS server -> client
70  connected [pseudo] [id]
71  response to a connect instruction
72  id: id of the player in the game
73 
74  ERRORS server -> client
75  error 1
76  corrupted ID
77  error 2
78  server is full (already 8 clients connected)
79  error 3
80  server is already playing
81  error 4

83  error 5
84  pseudo is too long (16 chars max)
85 */
